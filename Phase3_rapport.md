### 2. INGÉNIERIE TÉLÉCOM 360°
✅ **HLD/LLD Radio & Backhaul**
- HLD : roadmap 3-5 ans, stratégie spectre, dimensionnement CAPEX par phase
- LLD : paramètres physiques par site (GPS, azimuts, tilts), PCI planning, backhaul détaillé

✅ **SON (Self-Organizing Networks)**
- Auto-configuration : PCI auto-assignment, neighbour relation discovery
- Auto-optimisation : MLB, MRO, RACH optimization, CCO
- Tuning : handover (seuils RSRP/RSRQ), admission control, power control, scheduler

✅ **Dimensionnement CAPEX/OPEX**
- Modèle technico-économique : trafic projeté, capacité requise, nombre de sites
- Budget détaillé 3-5 ans : CAPEX (RAN, transmission, core), OPEX (spectrum, energy, maintenance)

✅ **SSV & Acceptance**
- Single Site Verification : tests RF, end-to-end, vérification KPI, PV Go/No-Go
- Intégration réseau : neighbour list, PCI conflicts, optimisation, monitoring 7 jours

### 3. DÉPLOIEMENT & INSTALLATIONS
✅ **HSE (Health, Safety, Environment)**
- Plan de sécurité : analyse de risques, EPI obligatoires, procédures d'urgence
- Conformité environnementale : EIE, gestion déchets, émissions RF ICNIRP
- Audit HSE : inspections hebdomadaires, non-conformités trackées, certifications ISO 45001/14001

✅ **Quality Gates multi-niveaux**
- Gate 1 : Site Survey validé (faisabilité, LoS, autorisations)
- Gate 2 : Génie civil & énergie achevés (fondations, abri, énergie, mise à terre)
- Gate 3 : Équipements installés (BTS/eNodeB, antennes, transmission, alimentation)
- Gate 4 : Acceptance technique (SSV, intégration, as-built, PV signé)

✅ **Dossiers as-built**
- Documentation technique : plans réalisés, photos, liste matériels, configurations exportées
- Traçabilité qualité : rapports de tests, certificats, fiches d'intervention, garanties

✅ **PV d'acceptance standardisés**
- Critères techniques : RSRP ≥-110 dBm à 67%, CSSR ≥98%, throughput ≥70% théorique
- Critères administratifs : conformité plans, délais, HSE 0 accident, documentation complète

### 4. SOLUTIONS IT & CYBERSÉCURITÉ
✅ **SOC 24/7**
- Architecture : SIEM centralisé, corrélation événements, Threat Intelligence, SOAR
- Équipe & processus : shifts 3x8, playbooks incidents, temps réponse (critique <15 min)
- Use cases : ransomware, accès non autorisés, DDoS, conformité PCI-DSS/ISO 27001

✅ **PRA/PCA**
- Stratégie : site principal + secours distant >50 km, réplication, bascule automatique
- RTO/RPO : Tier 1 (RTO <30 min, RPO <5 min), Tier 2 (RTO <2h, RPO <30 min)
- Composants : sauvegarde incrémentielle/complète, snapshots, stockage géoredondé

✅ **DevSecOps & Observabilité**
- Pipeline : SAST, SCA, DAST, policy-as-code, runtime protection
- Observabilité : métriques (RED), logs (ELK), traces (Jaeger), alertes (anomaly detection)
- SLI/SLO/SLA : disponibilité ≥99.97%, latence p95 <200ms

✅ **RTO/RPO contractuels**
- Tier 1 : RTO <30 min, RPO <5 min, disponibilité 99.99%
- Tier 2 : RTO <2h, RPO <30 min, disponibilité 99.97%
- Tier 3 : RTO <8h, RPO <4h, disponibilité 99.9%
