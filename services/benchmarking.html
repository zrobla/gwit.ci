Protocoles & méthodologie

Mesures ARTCI-compliant

Nos campagnes respectent strictement les spécifications techniques ARTCI avec protocoles traçables et équipements certifiés.

Méthodologie terrain

  • Routes stratifiées : urbain dense, périurbain, rural, axes routiers
  • Horaires critiques : 8h-10h (peak morning), 12h-14h (midday), 17h-19h (peak evening)
  • Échantillonnage : toutes les 2-5s selon vitesse véhicule
  • Technologies couvertes : 2G (GSM/GPRS/EDGE), 3G (UMTS/HSPA+), 4G (LTE/LTE-A), 5G NSA/SA

Équipements certifiés

  • Scanners multi-opérateurs Keysight/TEMS/Nemo
  • Smartphones de test calibrés (référence + DUT)
  • Serveurs test FTP/HTTP/Streaming dédiés
  • GPS haute précision (<5m de précision)

KPI par service

Scoring par couche

Évaluation granulaire selon pondération ARTCI : Voix 40%, Data 50%, SMS 10%.

Service VOIX (40% pondération)

  • Accessibilité : Call Setup Success Rate ≥98%
  • Maintien : Call Drop Rate ≤2%
  • Intégrité : MOS audio (POLQA/PESQ) ≥3.5/5
  • Délai : temps établissement <6s (2G/3G), <3s (4G/5G)

Service DATA (50% pondération)

  • Accessibilité RF : activation PDP/PDN ≥98%
  • Maintien RF : sessions non interrompues ≥97%
  • Accessibilité applicative : HTTP/FTP succès ≥95%
  • Débit : 4G DL ≥15 Mbps urbain, UL ≥5 Mbps
  • Latency : ping <50ms (4G), <20ms (5G)

Service SMS (10% pondération)

  • Taux succès : envoi/réception ≥98%
  • Délai bout-en-bout : <10 secondes
  • Scoring global : normalisation 0-100 avec seuils ARTCI
  • Heatmaps : géolocalisation par KPI et zone
